# إعداد OpenAI - للفهم الذكي للأسئلة

## لماذا OpenAI؟

حالياً النظام يستخدم **قواعد ثابتة** (rule-based) للرد على الأسئلة:
- ✅ يفهم أسئلة محددة مثل "كم عمرك؟" أو "بتدخن؟"
- ❌ لا يفهم أسئلة بصيغ مختلفة مثل "هل لديك حرارة؟" أو "سجل تاريخي بالأمراض؟"

مع **OpenAI**:
- ✅ يفهم أي صيغة للسؤال
- ✅ يرد بشكل طبيعي ومنطقي
- ✅ يحافظ على سياق المحادثة
- ✅ يتصرف كمريض حقيقي

---

## كيفية الإعداد:

### 1. احصل على API Key من OpenAI

1. اذهب إلى: https://platform.openai.com/
2. سجل حساب أو سجل دخول
3. اذهب إلى: https://platform.openai.com/api-keys
4. اضغط "Create new secret key"
5. انسخ الـ API Key (يبدأ بـ `sk-...`)

### 2. أضف الـ API Key لملف .env

افتح ملف `.env` وأضف:

```env
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 3. أعد تشغيل السيرفر

```bash
# أوقف السيرفر (Ctrl+C)
# ثم شغله من جديد
npm start
```

## كيف يعمل؟

1. **الطالب يسأل**: "هل لديك حرارة؟"
2. **النظام يرسل** السؤال + معلومات المريض لـ OpenAI

## بدون OpenAI:

- ✅ النظام سيعمل بشكل عادي
- ✅ سيستخدم القواعد الثابتة
- ⚠️ قد لا يفهم بعض الأسئلة

## أمثلة على التحسين:

### قبل OpenAI (Rule-based):
```
طالب: "هل لديك حرارة؟"
أفاتار: "معذرة، ممكن تعيد السؤال بطريقة أوضح؟" ❌

طالب: "سجل تاريخي بالأمراض؟"
أفاتار: "معذرة، ممكن تعيد السؤال بطريقة أوضح؟" ❌
```

### بعد OpenAI:
```
طالب: "هل لديك حرارة؟"
أفاتار: "حرارتي 37.2، طبيعية" ✅

طالب: "سجل تاريخي بالأمراض؟"
أفاتار: "نعم، عندي ضغط وسكري وكوليسترول عالي" ✅

طالب: "من متى وأنت مدخن؟"
أفاتار: "من 30 سنة، بدخن 20 سيجارة باليوم" ✅
```
